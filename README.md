# Case Manager - v14

این نسخه شامل اصلاح ظاهر آیکون پوشه‌ها و پرونده‌ها و رفتار Viewer مستندات است.

## تغییرات v14

- آیکون پوشه‌ها بازطراحی شد تا شبیه فولدر خاکستری با کاغذهای داخل آن باشد.
- هنگام hover روی پوشه، کاغذهای داخل فولدر با انیمیشن از داخل آن کمی بیرون می‌آیند.
- آیکون پرونده‌ها به شکل چند کاغذ/برگه طراحی شد.
- هنگام hover روی پرونده، کاغذها کمی بزرگ‌تر و برجسته‌تر می‌شوند.
- در بخش مستندات، کلیک روی عکس، ویدیو یا PDF یک Viewer بزرگ باز می‌کند.
- دکمه «بازگشت» در Viewer فقط همان بزرگ‌نمایی/نمایش فایل را می‌بندد و مودال پرونده باز می‌ماند.

## اجرا

```powershell
Remove-Item -Recurse -Force node_modules -ErrorAction SilentlyContinue
Remove-Item -Force package-lock.json -ErrorAction SilentlyContinue
npm install --legacy-peer-deps
npm run dev
```

اگر دیتابیس نسخه‌های قبلی را اجرا کرده‌ای، برای این نسخه SQL جدید لازم نیست.
